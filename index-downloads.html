Kernel X-Infinity
Kernel X-Infinity is a custom kernel, made by me. It aims to bring better battery life as well as better performance, with new features ofcourse.

Features

Governos

  • Bluactive
  • Smartmax
  • Darkness
  • Zzmoove
  • Intellidemand
  • Impulse
  • Performance
  • Lionfish
  • Interactive
  • Conservative
  • Ondeman
  • Userspace
  • Powersave
  • Interactive Pro
  • Intelliactive

Hotplugs

  • IntelliPlug
  • MSMHotPlug
  • ClusterPLug
  • BrickedPlug
  • ASMPlug
  • Zen Decision

I/O schedulers

  • noop
  • deadline
  • row
  • cfq
  • bfq
  • fiops
  • sio
  • sioplus
  • zen
  • tripndroid

Special Feactures

  • Double Tap Wake (DT2W)

More Feactures

  • Powered by Ubuntu 15.04 x86_64
  • LED notification support
  • And More

Supported Devices

  • Motorola Moto E 2015 LTE.
  • Moto G 2015.
  • Moto X Play.
  • Moto G 2015 Turbo Edition
